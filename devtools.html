NiM helps you debug your code with style and grace by leveraging Chromium DevTools and Node.js' debugging tool, the V8 Inspector. Further, the goal of this project is to improve the Node Developer's experience to the greatest extent possible. To that effort, this DevTools panel will be used to facilitate even greater NiM functionality as well as to support the ongoing efforts of its developers.
