NiM helps you debug your code with style and grace by leveraging Chrome DevTools and Node.js' debugging tool, V8 Inspector. Further, the goal of this project is to improve the Node developer's experience to the greatest extent possible. To that effort, this DevTools panel will be used to facilitate even greater NiM functionality.
